# Language Catalog

`languages_en.json` contains English display names for 693 languages and
language variants from the Unicode Common Locale Data Repository (CLDR).

- Source: https://github.com/unicode-org/cldr-json
- Source file: `cldr-localenames-full/main/en/languages.json`

The application bundles this catalog for its editable output-language selector.
Users can still enter a language name that is not present in the list.
